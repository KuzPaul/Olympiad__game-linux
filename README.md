# 🐧 Linux Olympiad Web (arcade)

Веб-платформа для проведения олимпиады и практикума по **Linux** в игровом формате.
Студенты проходят миссии на время, преподаватели управляют процессом, а тестировщики помогают отлаживать задания.

---

## ✨ Возможности

- 🎮 **5 миссий**
  классификация ПО, квиз по процессам, сопоставление прав, сборка сетевых команд, зомби‑процессы и bash

- ⏱ **Честный таймер**
  запускается **только после подтверждения** студентом готовности

- 🔁 **Разные правила попыток**
  - студент — **1 попытка** на тему
  - тестировщик — **без ограничений** (создаётся преподавателем)

- 🎵 **Фоновая музыка (Web Audio)**
  преподаватель может **отключить музыку для всех** одной кнопкой

- 📦 **Эталонные ответы**
  вынесены в `data/all_answers.json`
  генерация: `python3 scripts/export_answers.py`

## 📚 Учебные модули

- **Модуль 1 — Классификация ПО**  
  Разделение программ на системное, прикладное и сервисные утилиты.
- **Модуль 2 — Задачи и процессы**  
  Квиз по управлению процессами: состояния, сигналы, приоритеты, базовые команды.
- **Модуль 3 — Учётные записи и права**  
  Права доступа, числовые маски `chmod`, владение файлами и команды администрирования.
- **Модуль 4 — Сетевые средства Linux**  
  Практика по сетевым командам: `ip`, `ping`, `ss`, `curl`, `dig` и базовая диагностика.
- **Модуль 5 — Синтаксис Bash**  
  8 уровней, **1–2 строки** кода на уровень: shebang, присваивание, `$(( ))`, `$( )`, кавычки, цикл `for`, `exit`; финальный уровень засчитывается только при успешных предыдущих.

## 🚀 Быстрый старт

```bash
cd linux_olympiad_web_arcade
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python3 init_db.py
python3 app.py
```

Откройте в браузере адрес, который выводит Flask (по умолчанию `http://127.0.0.1:5000`).

### Учётная запись преподавателя (после `init_db`)

| Поле    | Значение  |
|---------|-----------|
| Логин   | `admin`   |
| Пароль  | `admin123` |

Смените пароль и `SECRET_KEY` в production.

### Переменные окружения

| Переменная   | Назначение        |
|-------------|-------------------|
| `SECRET_KEY` | Секрет сессий Flask |

## Структура репозитория

```
app.py                 # точка входа
init_db.py             # инициализация БД
olympiad/              # приложение Flask (модели, маршруты, настройки)
topics/                # данные тем (каждая тема в своей папке)
data/all_answers.json  # эталонные ответы (генерируются скриптом)
scripts/export_answers.py
static/                  # CSS, JS, опционально audio/
templates/               # шаблоны HTML
```

## Роли

| Роль        | Описание |
|------------|----------|
| `admin`    | Панель преподавателя, студенты, сброс паролей, **вкл/выкл музыки** |
| `student`  | Одна зафиксированная попытка на тему |
| `tester`   | Повторные проходы для отладки и тестирования |

Регистрация с сайта создаёт только студентов; тестировщиков добавляет преподаватель в админ-панели.
