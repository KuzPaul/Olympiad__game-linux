# Эталонные ответы

Файл `all_answers.json` генерируется из исходных тем в `topics/` и предназначен для **проверки заданий**, ревью и синхронизации с клиентом (в игре по-прежнему используются данные из `topics/*/topic.py`).

Обновить после изменения тем:

```bash
python3 scripts/export_answers.py
```

## Модуль сетевых команд (`builder`)

Для каждого шага в массиве `answers`:

| Поле | Смысл |
|------|--------|
| `task_index` | Номер задания по порядку (с нуля). |
| `scenario` | Текст условия, как в теме. |
| `primary` | Основной эталонный ответ (список токенов команды). |
| `accepted_variants` | Допустимые варианты (необязательно). |

## Модуль «Зомби и скрипт» (`zombie_script`)

Проверка **построчная**: каждая из 9 строк сравнивается с `primary` и `accepted_variants` своего шага. **Победа** — только если все строки засчитаны.

| Поле | Смысл |
|------|--------|
| `full_script_primary` | Справочный полный скрипт (сборка из эталонных строк). |
| `answers[]` | По одной записи на строку: `task_index`, `scenario`, `primary`, `accepted_variants`. |

Максимум **8 баллов**, если все 9 команд верны.
