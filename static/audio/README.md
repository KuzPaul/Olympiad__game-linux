# Аудио для миссий

Фон в миссиях подключается из этой папки через `/static/audio/…` (см. `static/js/topic_runner/music.js`).

Ожидаемые имена файлов:

- `music_topic_1.mp3` … `music_topic_5.mp3` — по одному на каждый модуль (тему) олимпиады  
  Если номер темы вне 1–5, в коде используется трек темы 1 (`music.js`).

Учитывайте авторские права на коммерческие композиции.
